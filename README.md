## Simulating
### Actual Mobile Simulation
I'm mainly using Android Studio's device emulator to test mobile views.
- https://docs.expo.dev/workflow/android-studio-emulator/

### Debugging on Physical Devices
To debug on a physical device, you can use Expo Go.
- Need to create an account and have a device connected to the same network as your development machine. (This is how I'm doing iOS testing with a Windows computer.)
- https://docs.expo.dev/workflow/expo-go/
